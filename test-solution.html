Our Approach to Test Solutions

Requirements Study

The input technical files are studied in detail and clarification are sought through Q&A and technical meetings. All the technical requirements are captured by Bofotia and approved the customer.

Design

The technical dossier are delivered for PDR/CDR as per requirements.

Procurement

Procurement is launched at the end of each specific Work Package as applicable.

Manufacturing

The manufacturing is performed as per the IPC standards. The cable harness is complied with EMC standard and IPC-620 and PCBs are complied with IPC-600/610. The labels and markings are complied with the labelling standards.

Integration

The integration of the bench is carried as per approved CDR documents.

Verification & Validation

The verification and validation is done as per IVVPr.

Delivery

The packaging and shipping follows international standard package requirements along with checklists.

Interface Test Adaptors (ITAs)

Interface Test Adaptor are used in conjunction with the test benches for testing specific LRUs. These Adaptors are manufactured based on Built to Spec or Built to Print. ITA Development

  • Study of Line Replaceable Unit (LRU) Test Specification
  • Study and configuration of test resources
  • Preparation of BOM, wire list, cable drawings based on the requirement
  • Documentation as per customer standards
Design, Development, Manufacturing cable harness for electromechanical assemblies
Complete ITAs
Designing of Interface Test Adaptor based on the ATE
Manufacturing
  • Manufacturing of self test cable for ATEs

Test Program Set

Bofotia undertakes development of Test Program Set (TPS) for Avionics Displays, Electronic Warfare, On board Computers, IFF and various other LRUs. The work involves System Architecting, Design, Coding, Integration and Testing of Test Program Set (TPS) for various LRUs.

Activities include – Test specification, Design, Coding of Test Software Interfacing with Data Acquisition System, Measuring Instruments and execution of Test Modules under different configuration for Testing.

LabWindows, LBMAPTE , C, XML are used in developing the test programs.

Bofotia has DGA Clearance.
TPS development experience for Thales, Teradyne, Schlumberger Test Benches using LabWindows, LabView, Test stand, Test Studio, CATE, Mediator and LASAR
